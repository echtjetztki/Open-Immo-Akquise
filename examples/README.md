# Referral Beispiele

Dieses Verzeichnis enthaelt Beispiel-Dateien fuer das Empfehlungsmodul.

## Dateien

- `referrals-example.json`: Beispiel-Datensaetze als JSON.
- `referrals-seed.sql`: SQL zum Einspielen von Beispiel-Daten in `public.referrals`.

## Verwendung

1. `referrals-seed.sql` in Supabase SQL Editor ausfuehren.
2. Im Dashboard `/referrals` neu laden.

Hinweis: Das SQL schreibt nur, wenn die Tabelle `public.referrals` existiert.
