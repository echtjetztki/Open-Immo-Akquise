# User Dashboard - Anleitung

## Übersicht

Das **User Dashboard** ist eine vereinfachte Ansicht für Mitarbeiter, die nur folgende Funktionen benötigen:

- ✅ Alle Properties ansehen
- ✅ Status bearbeiten
- ✅ Notizen hinzufügen und bearbeiten
- ❌ **KEINE** Bearbeitung der Property-Details (Kaufpreis, Link, etc.)

## Zugriff

**URL:** [https://open-akquise.vercel.app/user](https://open-akquise.vercel.app/user)

## Funktionen

### 1. Properties anzeigen

- Alle Properties werden als Karten angezeigt
- Jede Karte zeigt:
  - external_source ID
  - Übergeben am Datum
  - Kaufpreis (nur Ansicht)
  - Provision (nur Ansicht)
  - Aktueller Status
  - Notizen

### 2. Status bearbeiten

- Klicke auf "Bearbeiten" neben dem Status
- Wähle den neuen Status aus:
  - Übergeben
  - Kontaktiert
  - Besichtigung
  - Vermarktung
  - Verkauft
- Klicke auf "Speichern" ✓ oder "Abbrechen" ✗

### 3. Notizen verwalten

- Klicke auf "Bearbeiten" oder "Hinzufügen" neben Notizen
- Schreibe deine Notiz in das Textfeld
- Klicke auf "Speichern" oder "Abbrechen"
- Alle Notizen werden direkt in der Datenbank gespeichert

### 4. Filtern

- Nutze den Status-Filter oben, um nach Status zu filtern
- "Alle Status" zeigt alle Properties

### 5. Aktualisieren

- Klicke auf "Aktualisieren" um die neuesten Daten zu laden

## Navigation

In der Sidebar links:

- **Admin Dashboard** - Vollzugriff mit Statistiken (nur für Admins)
- **User Dashboard** - Deine Ansicht
- **Eingabe** - Neue Properties anlegen (nur für Admins)
- **Einstellungen** - Systemeinstellungen (nur für Admins)

## Hinweise

- Alle Änderungen werden sofort gespeichert
- Properties können **nicht gelöscht** werden über diese Ansicht
- Finanzielle Daten (Kaufpreis, Provision) sind **nur lesbar**
